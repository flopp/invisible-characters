Invisible Unicode characters?

In Unicode there are a lot of invisible, non-printable characters: regular white-space characters (e.g. {{charlink "0020"}}), language specific fillers (e.g. {{charlink "3164"}} of the Korean Hangual alphabet), or special characters (e.g. {{charlink "2800"}}). While all of these have a specific meaning in their natural context, they can be used in various applications that don't allow for regular whitespace characters.

Unusual Applications

While the popular game Among Us doesn't allow for an empty player name, or a name just containing regular space characters, you can copy and paste the U+3164 HANGUL FILLER when naming your player.

Send empty Whatsapp messages (instructions) using the {{charlink "200E"}}.

When copy and pasting the {{charlink "200C"}} into the title field of a Youtube video, the title becomes empty.

Send empty tweets (instructions) using the {{charlink "3164"}}.

You can have an invisible TikTok nickname (instructions!) by using the {{charlink "FFFC"}} character.

Send an empty Instagram comment (instructions) using the {{charlink "200B"}}.
